# 截图生成脚本使用说明

## 功能
自动生成20个不同分数的年龄偏好测试报告截图，用于测试和展示。

## 使用方法

### 1. 安装依赖
```bash
npm install
```

### 2. 启动开发服务器
在一个终端窗口中运行：
```bash
npm run dev
```

### 3. 生成截图
在另一个终端窗口中运行：
```bash
npm run screenshot
```

## 生成的截图

脚本会生成21张截图，覆盖0-100分的不同分数段：

- **0-20分**: 极度年下控 (6张截图)
- **21-40分**: 偏好年下 (4张截图)
- **41-60分**: 同龄最佳 (3张截图)
- **61-80分**: 偏好年上 (4张截图)
- **81-100分**: 极度年上控 (4张截图)

## 截图保存位置

所有截图保存在项目根目录的 `screenshots/` 文件夹中。

文件命名格式：`score-XXX-标签.png`

例如：
- `score-000-极度年下.png`
- `score-050-同龄.png`
- `score-100-极度年上.png`

## 测试模式

脚本使用URL参数来触发测试模式：
```
http://localhost:5173/?test=true&score=50
```

参数说明：
- `test=true`: 启用测试模式
- `score=0-100`: 指定要显示的分数

### 🔒 安全限制

测试模式**仅在本地开发环境**可用：
- ✅ `localhost`
- ✅ `127.0.0.1`
- ✅ `[::1]`
- ❌ 生产环境域名（自动禁用）

这确保了测试功能不会在生产环境被滥用。

## 注意事项

1. 确保开发服务器在 `http://localhost:5173` 运行
2. 截图生成过程中请勿关闭开发服务器
3. 生成过程大约需要1-2分钟
4. 截图为高清2x分辨率，文件较大
5. 测试模式仅在本地环境有效，部署后自动禁用
