# 📸 截图生成指南

## 快速开始

### 步骤1: 安装依赖
```bash
npm install
```

### 步骤2: 启动开发服务器
```bash
npm run dev
```
保持这个终端窗口运行。

### 步骤3: 生成截图（新开一个终端）
```bash
npm run screenshot
```

## 生成内容

脚本会自动生成 **21张** 不同分数的测试报告截图：

| 分数范围 | 类型 | 截图数量 |
|---------|------|---------|
| 0-20 | 极度年下控 | 6张 |
| 21-40 | 偏好年下 | 4张 |
| 41-60 | 同龄最佳 | 3张 |
| 61-80 | 偏好年上 | 4张 |
| 81-100 | 极度年上控 | 4张 |

## 截图位置

所有截图保存在：`screenshots/` 文件夹

文件命名示例：
- `score-000-极度年下.png` - 0分
- `score-050-同龄.png` - 50分
- `score-100-极度年上.png` - 100分

## 手动测试

你也可以在浏览器中手动测试任意分数：

```
http://localhost:5173/?test=true&score=75
```

将 `score=75` 改成 0-100 之间的任意数字即可查看对应结果。

## 技术细节

- 使用 Puppeteer 自动化浏览器
- 高清截图（2x分辨率）
- 全页面截图，包含所有内容
- 自动等待动画完成

## 故障排除

### 问题：找不到 puppeteer
**解决**：运行 `npm install` 安装依赖

### 问题：开发服务器未运行
**解决**：先运行 `npm run dev`，等服务器启动后再运行截图命令

### 问题：截图不完整
**解决**：检查网络连接，确保页面完全加载
